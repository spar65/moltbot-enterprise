# Auth0 Integration Guide Suite

This directory contains a comprehensive suite of guides for integrating Auth0 with Next.js applications. The guides are organized into multiple focused files to make it easier to find specific information.

## Guide Structure

1. **[Auth0 Guide Index](./00-Auth0-Guide-Index.md)** - Start here for an overview of all guides
2. **[Auth0 Setup Guide](./01-Auth0-Setup-Guide.md)** - Initial Auth0 tenant and application setup
3. **[Environment-Specific Guides](./02-Environment-Specific-Guides.md)** - Router-specific implementations
4. **[Advanced Auth0 Integration](./03-Advanced-Auth0-Integration.md)** - Advanced features and patterns
5. **[Auth0 Testing Guide](./04-Auth0-Testing-Guide.md)** - Testing strategies and debugging
6. **[Version Compatibility Guide](./05-Version-Compatibility-Guide.md)** - SDK version differences
7. **[Auth0 Implementation Checklist](./06-Auth0-Implementation-Checklist.md)** - Step-by-step implementation checklist

## Legacy Documentation

## Usage

1. Start with the [Auth0 Guide Index](./00-Auth0-Guide-Index.md) for an overview
2. Follow the [Auth0 Setup Guide](./01-Auth0-Setup-Guide.md) for initial configuration
3. Use the [Auth0 Implementation Checklist](./06-Auth0-Implementation-Checklist.md) to track your implementation
4. Refer to specific guides as needed for detailed information
